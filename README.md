jquery.socially
=====
A tiny jQuery plugin for dynamically adding social media share links to images

![screenshot](screenshot.png)

Usage:
-----

    $('img').socially();


Dependencies:
-----

- jQuery
- [jQuery-URL-shortener](https://github.com/hayageek/jQuery-URL-shortener)
